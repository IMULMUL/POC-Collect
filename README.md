## 为什么发起这个项目？
1. 几个月前，我参加了一场AWD攻防比赛，发现提前收集POC对比赛有好处而且在收集这些 `payload` 的同时，也能学到许多东西.
2. 后续的HW、CTF 等活动可以快速定位、利用漏洞.
